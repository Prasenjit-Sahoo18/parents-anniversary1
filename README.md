
# Parents' Anniversary Website

A modern, elegant web application celebrating a special milestone with interactive features, smooth animations, and an immersive user experience. Built with React, TypeScript, and Vite for optimal performance.

- **Theme Toggle**: Dark and light mode with smooth transitions

- **Animated Gallery**: Auto-cycling photo gallery with smooth transitions
- **Interactive Countdown**: Real-time countdown to the next anniversary
- **Guestbook**: Visitors can leave wishes and messages

- **Dark & Light Mode**: Toggle between themes for accessibility



### 3. Add Media Files

#### Background Music
Place your mp3 file in the `public` folder:

```powershell
Copy-Item "path/to/your/music.mp3" -Destination "public/Kesariya Tera Ishq Hai Piya-(Mr-Jat.in).mp3" -Force
```

Copy up to 5 photos to `public/photos/` directory:

```powershell
.\copy-images.ps1
```

### 4. Start Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`
